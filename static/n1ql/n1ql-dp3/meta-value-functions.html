Meta and value functions

Meta and value functions retrieve information about the document or item as well as perform Base64 encoding. The following table list the meta and value functions.

Function Description Returns
BASE64_VALUE(value) Computes the Base64 encoded value. This function could be used with non-JSON values stored in a bucket. Base64 encoded value
META() Retrieves the metadata for the document. Metadata object
VALUE() Retrieves the full value for the item in the current context. Value or NULL

Example

The following example shows the META() function via the N1QL tutorial.

SELECT META() FROM customer LIMIT 1

Returns:

{
  "resultset": [
    {
      "$1": {
        "id": "customer0"
      }
    }
  ]
}
